# Tableau.Airbnb_Turkey_Analysis
This project analyzes Airbnb listings in Istanbul using 2025 data to understand pricing dynamics, host quality, and revenue patterns across districts and room types.

The dataset was created by performing an inner join between Listings and Calendar data, followed by data cleaning and transformation in Tableau. Due to missing and null values in calendar price fields, the analysis primarily relies on listing-level price data.

-Key Questions Explored

How do average Airbnb prices vary across Istanbul districts?

How does room type affect pricing?

Do Superhosts receive higher guest ratings than non-Superhosts?

What is the relationship between guest capacity and estimated revenue?

How do price, reviews, and ratings interact with each other?

-Dashboard Highlights

Average Price by District (bar chart & map)
Reveals significant price differences across Istanbul districts, with central areas commanding higher prices.

Average Price by Room Type
Shows clear pricing distinctions between entire homes, private rooms, hotel rooms, and shared rooms.

Superhost Status vs Average Rating
Indicates that Superhosts receive slightly higher guest ratings on average.

Estimated Revenue vs Guest Capacity
Demonstrates diminishing returns beyond optimal capacity levels, especially for larger listings.
